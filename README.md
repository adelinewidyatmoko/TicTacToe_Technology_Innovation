# TicTacToe
Interactive TicTacToe game made using Flutter and Firebase.
Technology Innovation course final project in Information Systems major.

# Installation Requirements
- Dart and Flutter of versions specified in the pubspec.yaml must be installed beforehand
- In Project Settings -> Your apps -> Android in the Firebase Console, the valid SHA1 fingerprint must be
  - The SHA1 key is provided by running (from the project root)
    ```
    ./android/gradlew signingReport
    ```

    or (Linux):
    ```
    keytool -list -v -keystore ~/.android/debug.keystore -alias androiddebugkey -storepass android -keypass android
    ```
    or (Windows):
    ```
    keytool -list -v -keystore "%USERPROFILE%\.android\debug.keystore" -alias androiddebugkey -storepass android -keypass android
    ```


- `firebase init` must be run in the `tictactoe` directory
- A valid `firebase_options.dart` file generated by `flutterfire` is required within the `lib` directory

# Firestore Notes
## Mandatory Collections & Document Keys
A collection "leaderboard" containing at least one document with the boolean key-value ("hidden", true)

A hidden value of true hides the player from the leaderboard in the app
- "leaderboard":
    - hidden (bool)
    - score (number)
